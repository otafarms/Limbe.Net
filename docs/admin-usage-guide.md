# Admin Usage Guide

## Main Menu

Use `Limbe.Net Tourism` in WordPress admin to manage the platform.

Submenus include:

- Attractions
- Destinations
- Trip Ideas
- Partners
- Deals
- Events
- Booking Requests
- Settings
- Seed Importer

## Attractions

Create each attraction with:

- Title and main description.
- Featured image.
- Region, city, attraction type, and travel style terms.
- Ticket required status.
- Ticket price range. Use `Price not yet verified.` until confirmed.
- Official ticket URL or partner booking URL when available.
- Safety notice and advisory level.
- Last verified date or `Needs verification.`
- Source notes with official links where available.

The attraction single template displays quick facts, ticket information, safety notice, map placeholder, gallery, nearby planning, FAQ, booking CTA, and partner CTA.

## Destinations

Destinations are towns, regions, or travel hubs. Add overview, best-for notes, travel time placeholders, top attractions, where to stay, how to get there, safety notice, and last verified date.

## Partners

Partner listings support:

- Business type.
- Verified partner flag.
- Paid plan: Free, Verified, Featured, Premium.
- Sponsored content label.
- Listing expiry date.
- Booking URL and contact details.

## Booking Requests

Frontend forms create private `Booking Requests`. Review these in admin and follow up manually by email or WhatsApp.

## Settings

Configure:

- Default WhatsApp number.
- Default contact email.
- Google Maps API key placeholder.
- Enable or disable monetization CTAs.
- Currency: XAF, USD, or EUR.
- Affiliate disclosure.
- Safety disclaimer.

## Seed Content

Seed content is for launch scaffolding only. Review and verify all entries before promoting the site publicly.
