# SEO Launch Checklist

## Technical SEO

1. Set permalinks to `Post name`.
2. Install Rank Math or Yoast SEO.
3. Confirm custom post type archives are indexable.
4. Enable XML sitemaps.
5. Submit the sitemap in Google Search Console and Bing Webmaster Tools.
6. Add meta titles and descriptions for home, destinations, attractions, travel info, deals, and partner pages.
7. Confirm canonical URLs for translated pages.
8. Add `hreflang` through Polylang or WPML.
9. Compress uploaded images and add descriptive alt text.
10. Enable LiteSpeed Cache after verifying layout.

## Structured Data

The plugin outputs lightweight schema where possible:

- `TouristAttraction` for attractions.
- `LocalBusiness` for partners.
- `Event` for events.
- `FAQPage` for attraction FAQs.
- `BreadcrumbList` for tourism singles.

Use Rank Math or Yoast for broader site schema and sitemap controls.

## Content Quality

1. Add official links where available.
2. Add source notes for verification.
3. Keep last verified dates current.
4. Do not invent ticket prices.
5. Keep safety notices visible.
6. Label sponsored content and partner placements clearly.
7. Build internal links between destinations, attractions, itineraries, partners, and travel info.

## Pre-Launch Checks

1. Test homepage search.
2. Test attraction filters.
3. Test attraction single pages.
4. Test all lead forms.
5. Test mobile navigation.
6. Check Core Web Vitals after caching.
7. Crawl the staging site before pushing live.
